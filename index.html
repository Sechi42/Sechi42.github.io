Modelo de Predicción de Churn

Desarrollé y desplegué un modelo de machine learning en una página web de Streamlit para predecir churn de clientes utilizando Python, Docker, GitHub Actions, AWS EC2, ECR y FastAPI, logrando un AUC-ROC > 0.88.

Análisis de Sentimientos en Reseñas de IMDb con BERT

Construí y ajusté un modelo BERT para análisis de sentimientos en reseñas de IMDb, logrando un F1 score de 0.85 utilizando Python, BERT, Scikit-learn y Mlflow.

Predicción de Pedidos de Taxi

Desarrollé un modelo de machine learning para predecir la cantidad de pedidos de taxis en aeropuertos utilizando datos históricos. El modelo XGBoost fue el más eficiente, logrando un RMSE de 40.78, superando el objetivo planteado (RMSE ≤ 48).
